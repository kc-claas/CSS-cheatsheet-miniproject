Keith's CSS Cheatsheet

Here's some prepared CSS syntax for quick reference

Flexbox Row with Wrap

This will create a flexbox row layout with wrapping 

    element{display: flex;
        flex-direction: row;
        flex-wrap: wrap;}

Flexbox Column

This will create a flexbox column layout 

    element{display: flex;
        flex-direction: column;}

Media Query

This will create a media query to modify css on screen width under 768px 

    @media screen and 
    (max-width=768px) element{}

Background Gradient

This will create a background 'image' of a linear gradient 

    element{
        background-image: 
        linear-gradient:
        (color1 0%, color2 100%);}

Border and Shadow Transition

This will make borders and box shadows change when the object is hovered over 

    element{transition: 
        all 0.5 ease-in-out}
                    
    element:hover {
        border: solid white 3px;
        box-shadow: 
        inset 0px 0px 5px color1;}

Title Card Positioning

This will align title cards to be overlap with the top border of the tiles 

    element{position:absolute;
        top: -50px;}
